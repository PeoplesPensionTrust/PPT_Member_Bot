
context_documents folder contains the pdf documents used for context for the RAG portion of the bot.

asset_generation_training_data contains the data required to build the embeddings and the FAISS index

generated_assets contain the built index and embedding files.

requirements.txt contains all required libraries and their versions.

api_key folder has a .txt file called "api_key.txt". Paste your OpenAI API key in the first line of that file.
